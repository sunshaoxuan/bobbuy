# BOBBuy

**BOBBuy** 是一款全球委托任务（Errand Service）平台，旨在链接有代办需求的用户与能够提供服务的个人或团队。

## 📍 初步场景：东京 Costco 代购
作为平台的切入点，BOBBuy 首先解决身处海外或异地用户对 Costco 商品的获取障碍。通过智能撮合与流程数字化，提高代购效率与信任度。

## ✨ 核心特性
- **💬 即时通讯 (IM)**：集成 AI 助手的实时聊天系统，支持文本、图片收发及自动意图解析。
- **🤖 AI 智能代理**：
  - **意图识别**：自动解析对话内容，整理购物清单。
  - **OCR 识别**：读取购物小票及商品标签，自动录入价格与库存信息。
  - **清单建议**：根据历史数据提供智能推荐。
- **🧾 订单自动化**：支持复杂的多客户订单拆分、费用平摊及自动核销。
- **🌐 全球化支持**：初期支持中日双语，松耦合架构易于全球各区域扩展。

## 🛠 技术架构
- **前端**：移动优先的 H5 网页（Vue/React），未来支持原生 iOS/Android App。
- **后端**：微服务架构，基于 Python/Node.js/Go（待选型）并部署于云端。
- **存储**：关系型数据库 (PostgreSQL) + 缓存 (Redis) + 对象存储 (S3 兼容)。
- **AI 系统**：以 LLM (GPT-4 级别) 为核心的任务代理调度系统。

## 🚀 路线图
1. **Phase 1 (MVP)**：完成核心代购闭环流程（H5版）。
2. **Phase 2**：推出原生移动端应用，强化 AI 对话与物流路径优化。
3. **Phase 3**：全面扩展至全球其他城市及更多生活服务品类。
