# ARCH-07: 利润核算与经营分析详细设计

## 1. 模块概述
本模块负责对行程结束后的整体经营状况进行财务复盘。通过聚合 [ARCH-06](ARCH-06-最终结账与线下结算详细设计.md) 的实收数据与商家输入的经营成本，计算出单次行程的精确净利润。

## 2. 经营成本体系 (Cost Management)
商家在“财务看板”中可增量记录以下经营支出：
-   **固定支出**：本次行程的能源消耗（油/电）、停车费、路桥费。
-   **物料支出**：分装所需的包装袋、耗材费用。
-   **杂项支出**：现场餐补或其他不可计入订单的杂项费用。

## 3. 利润结算模型 (Profit Model)
-   **毛收益 (Gross Revenue)**：所有订单中提取的 `代购费 + 服务费` 总和。
-   **总成本 (Total Cost)**：所有输入的经营支出总和。
-   **净利润 (Net Profit)** = `毛收益 - 总成本`。

## 4. 统计报表预留
-   **单程报表**：收支对比图表（饼图/条形图）。
-   **多程趋势**：月度/季度利润波动曲线，帮助商家优化选品与行程密度。

## 5. UI 布局建议
-   **财务分析页**：大字报风格的净利润数字展示，下方列出详细的收支清单。
